Data 1: Fundamentals of data

This course is about the fundamentals of working with data: how to get it, understand it, check its quality, tidy it, visualise it, analyse it, and present the results.

There are four modules, each of which has nine topics.

Module 1: Data fundamentals

Module 1 is about data fundamentals. It explains what data is; why data is useful; what the main data types and storage formats are; how to understand a data set; how to assess its quality; and how to prepare it for use.

Module 2: Visualising data

Module 2 is about visualising data. It explains what visualisation is; what the principles are that underly it; why it is so powerful; what the most commonly used visualisations are; and when you might use one rather than another.

Module 4 is about using data responsibly. It explains how to respect data ownership; how to handle personal information with care; how to avoid being misled by some of our most common biases and fallacies of reasoning; and how to present data honestly and clearly.
